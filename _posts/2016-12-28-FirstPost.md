Let's be honest: this site's primary function is to add another link to my resume and contact information so that I can find a job suited to my talents and interest. Fortunately, Barry Clark's freely available Jekyll Now theme [source code](https://github.com/barryclark/jekyll-now) and [tutorial](https://www.smashingmagazine.com/2014/08/build-blog-jekyll-github-pages/) helped this statistician quickly (and enjoyably) produce a website for that purpose.

Fun fact: as a teacher at [Lake Mills School]() I was asked to take over a 1-semester Computer Science class that had been written by a colleague. I found the content to be different than what I had expected; it included taking apart donated electronics, a module in HTML and CSS and even a few days learning about EBay (!!). I made some substitutions and included several weeks on functional programming in Python, but the students enjoyed HTML the most :)
